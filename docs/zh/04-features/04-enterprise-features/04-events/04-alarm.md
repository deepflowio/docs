# 告警

告警页面可以记录系统中的告警事件。例如，当系统检测到安全漏洞、异常行为或其他需要用户关注的情况时，会生成对应的告警事件，并提供相应的详细描述和建议。

用户可根据策略等级、事件等级、监控对象、创建人、监控对象、区域、事件UID等多维度进行过滤查询。

- 可通过表格的`列选项`功能，添加要查看的事件信息，使用详情，请参阅【[应用 - 调用日志](../07-application/05-call_log.md)】章节
