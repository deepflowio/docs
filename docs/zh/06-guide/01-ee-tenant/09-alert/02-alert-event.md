# 告警事件

告警事件页面中以图表的方式将告警策略数据进行可视化，可从多维度对告警事件进行分析。

![02-alert_event.png](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/20230428644b440666e6f.png)

- 事件列表：展示告警策略生效后获取到的事件信息
  - 操作
    - 查看详情：以弹框的形式展示对应告警事件的详细信息
    - 删除：删除告警事件
