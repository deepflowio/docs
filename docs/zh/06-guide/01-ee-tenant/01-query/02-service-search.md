# 资源搜索框

应用-资源统计、网络-资源统计、网络-资源盘点都使用的是`资源搜索框`。

![服务搜索框](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/2024031965f948c36d189.png)

- **① 切换搜索框**：快速切换搜索框，支持自行输入搜索内容，或固定的搜索项以供用户选择。使用详情，请参阅【切换搜索框】章节
- **② 自由搜索输入框**：通过输入`搜索标签`来进行数据的搜索查询，搜索框支持中英文联想输入，搜索标签的使用，请参阅【搜索标签】
- **③ 切换主分组**：资源分组，对应功能界面的`资源`
- **④ 切换次分组**：其他分组，对应功能界面的`分组属性`
- **⑤ 保存搜索条件**：保存当前界面输入的搜索条件
- **⑥ 搜索按钮件**：点击后触发页面查询，可通过系统设置来配置页面查询触发方式，按钮使用详情，请参阅【[设置- 搜索框配置](../../configuration/settings/)】

## 切换搜索框

支持切换自由搜索、容器搜索、进程搜索三种搜索模式，其中，相较于自由搜索，容器搜索与进行搜索不仅支持自由搜索输入框，同时也支持固定搜索输入框，接下来将分别介绍三种模式。

### 自由搜索

自由搜索下，可根据需求自行添加搜索标签。

![自由搜索](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/2024032065fa5c504e8b2.png)

- **① 切换搜索框**：可切换到`容器搜索`或`进程搜索`
- **② 自由搜索输入框**：通过输入`搜索标签`进行数据的查询，搜索框支持中英文联想输入。搜索标签的使用，请参阅【搜索标签介绍】

### 容器搜索

容器搜索由`固定搜索输入框`与`自由搜索输入框`组成，`固定搜索输入框`将部分容器相关的 Tag 字段固定，可快速选择查询的数据，提升搜索效率。

![容器搜索](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/2024032065fa8e5d80f03.png)

- **① 固定搜索输入框**：下拉选择需查询的数值，即`固定 Tag = 选中值`
  - **查询逻辑:** 
    - `固定 Tag`之间使用 and 连接
    - `固定 Tag`与`自由搜索 Tag` 之间使用 and 连接
- **② 自由搜索输入框**：使用详情，请参阅【自由搜索】章节
- **③ 自由搜索收起**：展开/收起自由搜索输入框
- **④ 分组**：容器搜索仅支持主分组，若所在页面不支持分组查询，则不支持分组

### 进程搜索

进程搜索由`固定搜索输入框`与`自由搜索输入框`组成，`固定搜索输入框`将部分容器相关的 Tag 字段固定，可快速选择查询的数据，提升搜索效率。

![进程搜索](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/2024032065fa975e8150e.png)

- **① 固定搜索输入框**：使用详情，请参阅【容器搜索】章节
  - **查询逻辑:** 
    - `固定 Tag`之间使用 and 连接
    - `固定 Tag`与`自由搜索 Tag` 之间使用 and 连接
- **② 自由搜索输入框**：使用详情，请参阅【自由搜索】章节
- **③ 自由搜索收起**：使用详情，请参阅【容器搜索】章节
- **④ 分组**：使用详情，请参阅【容器搜索】章节

## 搜索标签

每一个完整的搜索条件称之为一个`搜索标签`，接下来详细介绍`搜索标签`。

### 搜索标签介绍
 
![搜索标签展示](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/2024031965f963331e436.png)

- **① Tag 名称**：查询的字段名称，不同页面支持搜索的字段不同，可在页面的`设置 -> 数据库字段`可查看支持搜索的 Tag 字段。Tag 字段搜索演示，请参阅【Tag 字段搜索演示】章节
  - **语义:** 不同的 Tag 之间使用 and 连接，相同 Tag 之间根据`操作符`的不同使用的逻辑运算符不一致
    - a：`=`、`:`、`~` 用 or 连接
    - b：`!=`、`!:`、`!~` 用 and 连接
    - c：`>=`、`<=`、`>`、`<` 用 and 连接
    - a/b/c 连接后，再使用 and 连接，例如：`搜索条件输入框：server_port > 20, server_port < 80, server_port != 44, server_port != 45`，生效的条件为`(server_port > 20 and server_port < 80 ) and (server_port != 44 and server_port != 45)`
- **② 操作符**：输入正确 Tag 字段后，自动展示可使用的操作符。目前支持精准匹配、模糊匹配、正则匹配
  - 精准匹配：对应`=`、`!=`、`>=`、`<=`操作符，对于`resource 类型`的 Tag 按资源 ID 进行精准匹配，其他则按实际输入进行精准匹配
  - 模糊匹配：对应`:`、`!:`操作符，字符串匹配，支持`*`通配符。例如`*123*`则匹配所有<mark>包含</mark>`123`的字符串，而`123`则只匹配<mark>完全等于</mark>`123`的字符串
  - 正则匹配：对应`~`、`!~`操作符，字符串正则匹配
- **③ Tag 值**：根据`Tag 名称`及`操作符`展示可选择的值或者直接输入需要使用的值
  - NULL：空值，一般配合`!=`使用，为过滤`全部`的语义
  - ⑦ 表格过滤：当筛选候选项出现重名的情况时或者需要多选时，可通过`表格过滤`精准资源
- **④ 禁用**：禁用当前`搜索标签`对应的搜索条件
- **⑤ 修改**：修改当前`搜索标签`对应的搜索条件
- **⑥ 删除**：删除当前`搜索标签`
  
### Tag 字段搜索演示

![Tag 字段搜索演示](https://yunshan-guangzhou.oss-cn-beijing.aliyuncs.com/pub/pic/2024031965f9633407d83.png)

- **① 输入字段**：在搜索框中输入要查询的 Tag 字段，支持中英文联想匹配 
- **② 展示匹配字段**：展示所有满足条件匹配到的 Tag 字段
- **③ Tag 字段详细信息**：可查看鼠标所在的 Tag 相关信息，如类别、类型、分类、描述信息等
